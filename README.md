# OgUltra Panel - Official Images

Official image/egg collection for **OgUltra Panel** (fork of Draco/Teryx/Skyport).

## 📦 Seed File

Import all images at once using the seed URL:

```
https://raw.githubusercontent.com/ogultra-dev/images/main/seed/ogultra-seed.json
```

Replace `YOUR_USERNAME` with your GitHub username after uploading.

## 🚀 Setup

1. Fork or clone this repo to your GitHub account.
2. Replace `YOUR_USERNAME` in the seed file with your actual GitHub username.
3. Go to OgUltra Panel → Images → Import Seed.
4. Paste the seed URL.
